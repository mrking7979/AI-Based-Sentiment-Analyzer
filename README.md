# 🤖 AI-Based Sentiment Analyzer

An intelligent AI-powered web application that analyzes user sentiment from text using Hugging Face Transformers and stores analysis history in MongoDB with dynamic visualization reports.

---

# 🚀 Features

✅ AI-Based Sentiment Analysis  
✅ Positive / Negative Detection  
✅ Confidence Score Prediction  
✅ Dynamic Animated UI  
✅ Glassmorphism Design  
✅ MongoDB Database Storage  
✅ Sentiment History Tracking  
✅ Pie Chart Visualization  
✅ Real-Time Analysis  
✅ Flask Backend Integration  

---

# ⚙️ Installation

## 1️⃣ Clone Repository

```bash
git clone <your-github-link>
```

---

## 2️⃣ Open Project Folder

```bash
cd AI-Based-Sentiment-Analyzer
```

---

## 3️⃣ Install Dependencies

```bash
pip install flask transformers torch pymongo pandas matplotlib
```

---

# 💾 MongoDB Setup

1. Install MongoDB Community Server
2. Open MongoDB Compass
3. Connect to:

```bash
mongodb://localhost:27017/
```

---

# ▶️ Run Project

```bash
python app.py
```
